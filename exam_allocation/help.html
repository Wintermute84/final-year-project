Shuffle Types and Grouping for 1 Semester

Shuffle Type format for a single semester.

  • Side A and Side B represents the left and right seat of a desk respectively.
  • The students at the top of side A will be automatically allocated to the drawing room, so place necessary branches (eg. Computer Science Streams) in side A.
  • Any class having the same name should be consecutive (eg. CSEA -> CSEB)
  • The classes which should not share the same desk should be consecutive (eg. AI -> AIDS).
  • Try to fill out computer section on one side and the non computer section on the other side.
  • For classes sharing the same elective, try to fill them out at the same side.
